Project Name: Assistant Tour Guide Robot

Project Overview:
The project is called Assistant Tour Guide Robot. It is a smart robotic system designed to help visitors inside museums by providing guided tours, artifact explanations, and interactive question answering.

Project Goal:
The main goal of the project is to create an assistant robot that can guide museum visitors, explain historical artifacts, answer visitor questions, and improve the museum experience using artificial intelligence, speech interaction, and a user-friendly touch screen interface.

Main Idea:
The robot works as an interactive museum tour guide. Visitors can choose the language and tour type from the screen. The robot then explains selected artifacts using voice output and allows visitors to ask questions using speech or typed input. The system can use local museum data and an AI model to generate suitable answers.

Project Purpose:
The purpose of the Assistant Tour Guide Robot is to make museum tours more interactive, accessible, and engaging. It supports visitors by giving information in a clear and simple way, helping them learn about artifacts without always needing a human guide.

Short Description:
The Assistant Tour Guide Robot is a smart bilingual museum guide robot that combines a graphical user interface, speech recognition, text-to-speech, artificial intelligence, and robot navigation to support visitors during museum tours.
